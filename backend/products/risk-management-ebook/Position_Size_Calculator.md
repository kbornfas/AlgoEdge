## POSITION SIZE FORMULA

```
Position Size (Lots) = Risk Amount / (Stop Loss in Pips × Pip Value per Lot)

Where:
- Risk Amount = Account Balance × Risk %
- Pip Value = ~$10 for major pairs (1 standard lot)
```

### Example Calculation:

**Given:**
- Account: $5,000
- Risk: 2% = $100
- Stop Loss: 40 pips
- Pip Value: $10/pip

**Calculation:**
```
Lots = $100 / (40 × $10) = $100 / $400 = 0.25 lots
```
