## CHART SETUP

### Recommended Settings
- Timeframe: H1 or H4 for swing trading
- Chart Type: Candlesticks
- Grid: Off
- Ask Line: On (helps see spread)

### Quick Chart Cleanup
1. Right-click → Properties → Common
2. Uncheck: Grid, Period Separators
3. Check: Show Ask Line

## POSITION SIZING FORMULA

```
Lots = (Balance × Risk %) ÷ (SL in Pips × Pip Value)

Example:
$10,000 × 1% = $100 risk
50 pip SL × $10 pip value = $500
Lots = $100 ÷ $500 = 0.20 lots
```
