## REVERSAL PATTERNS

### Head & Shoulders (Bearish)
```
      Head
       /\
      /  \
  LS /    \ RS
   /\    /\
  /  \  /  \
------ NECKLINE ------
```
- Entry: Break below neckline
- Stop: Above right shoulder
- Target: Head height from breakout

### Inverse Head & Shoulders (Bullish)
```
------ NECKLINE ------
  \  /  \  /
   \/    \/
  LS \  / RS
      \/
      Head
```
- Entry: Break above neckline
- Stop: Below right shoulder
- Target: Head height from breakout

### Double Top (Bearish)
```
   /\    /\
  /  \  /  \
 /    \/    \
------ NECKLINE ------
```
- Entry: Break below neckline
- Stop: Above second top
- Target: Pattern height from breakout

### Double Bottom (Bullish)
```
------ NECKLINE ------
 \    /\    /
  \  /  \  /
   \/    \/
```
- Entry: Break above neckline
- Stop: Below second bottom
- Target: Pattern height from breakout

## CONTINUATION PATTERNS

### Bull Flag
```
    |
    |  ----
    | /   /
    |/   /
    |   /
    |  /
    | ----
    |
```
- Pole + downward sloping channel
- Entry: Break above flag resistance
- Target: Pole length from breakout

### Bear Flag
```
    |
    | ----
    |  \   \
    |   \   \
    |    \  
    |     ----
    |
    |
```
- Pole + upward sloping channel
- Entry: Break below flag support
- Target: Pole length from breakout

### Ascending Triangle (Bullish)
```
-----------
    /\  /\
   /  \/
  /\  /
 /  \/
/
```
- Flat resistance + rising support
- Entry: Break above flat line
- Target: Triangle height from breakout

### Descending Triangle (Bearish)
```
\
 \  /\
  \/  \
  /\   \
 /  \  /\
-----------
```
- Flat support + falling resistance
- Entry: Break below flat line
- Target: Triangle height from breakout

### Symmetrical Triangle
```
\         /
 \   /\  /
  \ /  \/
   X
  / \  /\
 /   \/  \
/         \
```
- Converging trendlines
- Entry: Break either direction
- Target: Widest part from breakout

## PATTERN TRADING RULES

### Entry Rules
1. Wait for candle CLOSE beyond level
2. Volume should increase on breakout
3. Retest of breakout level adds confidence

### Stop Loss Rules
1. Beyond pattern's opposite extreme
2. Allow for false breakout buffer (10-20 pips)
3. Never wider than pattern height

### Target Rules
1. Minimum: Pattern height from breakout
2. Adjust for S/R levels in the way
3. Consider partial exits at key levels

## PATTERN CHECKLIST

Before trading any pattern:
□ Pattern is clearly visible
□ Proper structure (correct touches)
□ Trend context (with or against?)
□ Volume confirms breakout
□ Risk/reward minimum 1:1.5
□ No major news imminent

## COMMON MISTAKES

❌ Trading before pattern completes
❌ Ignoring trend direction
❌ Not waiting for breakout close
❌ Setting target beyond major S/R
❌ Taking every pattern regardless of context
