# CLAUDE.md

This file documents the use of Claude AI in the AlgoEdge project.

## Purpose
Claude is an AI assistant used for code generation, troubleshooting, and automation tasks within the AlgoEdge workspace.

## Usage
- Run Claude from the terminal using the `claude` command.
- Use Claude for:
  - Generating code snippets
  - Debugging errors
  - Automating repetitive tasks
  - Answering technical questions

## Troubleshooting
If the `claude` command fails (exit code 1):
- Check your installation and environment variables.
- Ensure you have network access if Claude requires cloud connectivity.
- Review terminal output for error details.

## Notes
- Claude is intended to complement GitHub Copilot and other AI tools.
- For best results, provide clear and specific prompts.
